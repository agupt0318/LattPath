# Legacy Prototype

This directory is retained as historical reference material from the older Visual Studio prototype.

It is **not** the supported entrypoint for the repo.

If you want to build, run, or test the maintained planner, use:

- `src/`
- `include/`
- `CMakeLists.txt`
- the `lattpath` CLI produced by the CMake build

Why this folder still exists:

- it preserves the original prototype code for comparison
- it provides historical context for the project’s evolution
- it can be useful when tracing older ideas or assets

What this folder is not:

- it is not covered by the current CMake build
- it is not the target of the current CI workflow
- it is not the recommended place to add new features or fixes

Treat it as archived reference code, not as the maintained planner implementation.
